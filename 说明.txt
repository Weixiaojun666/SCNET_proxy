项目地址 https://github.com/Weixiaojun666/SCNET_proxy 新版本可在此页面获取

本项目基于JDK20
配置文件详解

[setting.json]

{
  "connection_limit": 1,   //限制单IP建立连接数 【设置为1可以防小号同时在线】
  "time_out": 8,//超时时间 过小会导致频繁掉线，过大会导致内存不能及时释放 [单位 秒]
  "cache_time": 60,//服务器信息缓存时间，可以有效减少打到服务器上的查询请求 [单位 秒]
  "token": "",//WeiServers Token 可在群机器人内查看自己的Token
  "timeout_retry":5,//超时重试次数 默认为5 过大会导致程序卡慢 过小可能会导致玩家检查失败
  "cloud": true,//是否开启WeiServers云服务。[在没有有效TOKEN时此项自动停用]
  "verification": {
  "basis":true,//是否开启WeiServers玩家验证。[区域封禁 黑白名单 以及高级验证]
  "qq_bind":true //验证是否绑定了qq
  "Verified":true //验证是否已实名
  }
  "whitelist": false,//开启后服务器仅限本地白名单可进入
//服务器列表 
  "server_list": [
    {
      "name": "测试",//name为服务器名称[启用云服务后可以用来下方指令和查看统计,重复必出BUG]
      "address": "127.0.0.1", //address[一般填127.0.0.1即可]
      "port": 28887,//port填服务器的端口
      "proxy_port": 30000//proxy_port为对外开放的端口
    }
  ]
}

[whitelist]
[{
  "id": "22473", //玩家社区ID 以社区id进行校验
  "name": "Weixiaojun" //玩家社区昵称 这个不做校验,只是给服主自己看的
}]

实际上目前判断只以ID为主 name判断目前没做！
白名单目前只支持手动增加，指令暂时未做！

配置文件为严格的json格式 不支持注释

目前区域封禁列表和黑白名单列表为C7和Weixiao在维护，将会在完工后开放修改权限。
